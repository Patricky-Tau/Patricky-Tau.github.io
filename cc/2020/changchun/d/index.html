CC2020-ChangChun-D

题意

定义函数

\[ a_n = \begin{cases} 1, & n = 0 \\ c \cdot \max\limits_{0 \le i < n} a_{n \operatorname{\&} i}, & \text{otherwise} \end{cases}, \]

求 \(\left( \displaystyle\sum\limits_{i=0}^n a_i \right) \bmod (10^9+7)\)

https://codeforces.com/gym/102832/problem/D

  • \(0 \le n \lt 2 ^ { 3000 }\)
  • \(0 \le c \le 10 ^ 9\)

解答

尝试手推 \(\{a\}\) 的前几项（\(c = 2\)）：

下标 \(0\) \(1\) \(2\) \(3\) \(4\) \(5\) \(6\) \(7\)
二进制 \(0\) \(1\) \(10\) \(11\) \(100\) \(101\) \(110\) \(111\)
数值 \(1\) \(2\) \(2\) \(4\) \(2\) \(4\) \(4\) \(8\)

即 \(a_i = c ^ {\operatorname{popcount}(i)}\)。 首先注意到 \(n\) 全一的情况是二项式定理的经典形式：

\[ \sum_{i = 0}^n \binom{n}{i} p ^ i = (1 + p) ^ n \]

而如果有某些位不是 \(1\)，则可以按位拆分为多个上述的和式，即对于所有 \(n_i = 1\) 的位：

\[ \verb|calc|(i) = \left( (c + 1) ^ {N - i - 1} + c \times \verb|calc|(i - 1) \right) \]
